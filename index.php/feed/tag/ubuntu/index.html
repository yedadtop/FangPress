现在的宝塔臃肿有广告，我只能选择老版本，又怕网上下的有后台....于是我在github上找到了老版本。
github上的项目地址：https://github.com/8838/btpanel-v7.7.0

安装链接：

curl -sSO https://gitee.com/yedadtop/bt7.7.0/raw/master/install/install_panel.sh && bash install_panel.sh
curl -sSO http://www.yedad.top/main/install/install_panel.sh && bash install_panel.sh
curl -sSO http://10.0.0.6/main/install/install_panel.sh && bash install_panel.sh

面板卸载

curl -sSO https://gitee.com/yedadtop/bt7.7.0/raw/master/btuninstall.sh && bash btuninstall.sh
curl -sSO http://www.yedad.top/main/btuninstall.sh && bash btuninstall.sh
curl -sSO http://10.0.0.6/main/btuninstall.sh && bash btuninstall.sh

优化指令

1.一键优化(解锁专业版插件，去倒计时，延时等待，关闭客服活动推荐，去除强制绑定等等等等......)：

curl -sSO https://gitee.com/yedadtop/bt7.7.0/raw/master/mendbt.sh && chmod +x mendbt.sh && bash mendbt.sh
curl -sSO http://www.yedad.top/main/mendbt.sh && chmod +x mendbt.sh && bash mendbt.sh
curl -sSO http://10.0.0.6/main/mendbt.sh && chmod +x mendbt.sh && bash mendbt.sh

2.给plugin.json文件上锁防止自动修复为免费版。注意！这会导致文件不可被修改

chattr +i /www/server/panel/data/plugin.json

3.解锁

chattr -i /www/server/panel/data/plugin.json

注意

1.解锁宝塔所有付费插件为永不过期的原理：（优化指令有）
文件路径：/www/server/panel/data/plugin.json搜索字符串："endtime": -1全部替换为"endtime": 999999999999
2.需要手动打开/www/server/panel/data/plugin.json文件之后再使用mendbt.sh才能成功破解付费插件。
