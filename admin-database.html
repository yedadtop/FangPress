导出数据库

将整个数据库（所有表 + 索引 + 数据）导出为 .sql 文件。 导出文件可用于备份、迁移或在导入页中恢复。

导入 SQL

SQL 一旦执行将立即生效，导入完成后会自动清理全站缓存。执行多条语句时会作为一个事务串行执行，若任一条失败将终止后续语句。

重要：导入过程缺乏端到端原子性

受 Cloudflare 后端限制，跨批次之间无法构成一个事务，每一批（50 条）虽然在数据库内部是原子的，但批次之间相互独立。若在导入过程中发生网络中断、浏览器关闭或后端异常，已成功提交的批次无法回滚，数据库可能留下脏数据。建议在导入前先导出当前数据作为备份，并优先在网络稳定的环境下执行。
